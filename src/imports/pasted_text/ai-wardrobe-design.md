Design a complete modern SaaS Web Application UI/UX for an AI-Powered Smart Wardrobe Management System.

PROJECT OVERVIEW

The Smart Wardrobe Management System is an AI-powered fashion management platform that helps users:

* Upload clothing images
* Scan clothing using AI
* Automatically detect clothing categories
* Organize personal wardrobes
* Manage wardrobe zones
* Store fashion preferences
* Join friend groups with similar interests
* Receive AI-powered outfit recommendations
* Receive event-based recommendations
* Receive collaborative filtering recommendations based on similar users and friend groups

The system follows a Microservice Architecture consisting of:

* User Service
* Wardrobe Service
* AI Service
* Recommendation Service
* Storage Service
